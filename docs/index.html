dcl is a micro library written in JavaScript for node.js and browsers that implements OOP with mixins + AOP at both "class" and object level.

Module documentation

This documentation is a work in progress.

  • mini.js - minimal dcl kernel, you might want to use it on browsers to conserve bandwidth
    • dcl() - the composition engine
    • superCall() - super call decorator
    • mix() - mix in one object with another
    • delegate() - delegate from one object to another
  • dcl.js - full dcl kernel that includes chaining control and AOP support
    • advise() - general advice decorator
    • before() - "before" advice decorator
    • around() - "around" advice decorator
    • after() - "after" advice decorator
    • chainBefore() - "before" chaining directive
    • chainAfter() - "after" chaining directive
  • advise.js - object-level AOP with dynamic advisement
    • advise() - general advice directive
    • before() - "before" advice directive
    • around() - "around" advice directive
    • after() - "after" advice directive
  • inherited.js - dynamic super calls without decorators
    • inherited() - super call
  • debug.js - error-checking and introspection facility
    • log() - log information of classes and objects
